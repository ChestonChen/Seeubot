Seeybot 把 MacBook 刘海变成 AI 编程会话的实时看板 —— 多少在 工作、多少 空闲、一共烧了多少 token。

它能做什么

一个小巧的
AI 会话指挥中心。

所有数据都从 Claude Code 和 Codex 本地写好的文件读取。不联网、无需 API Key、无任何遥测。

📊

实时看板

会话数、工作 vs 空闲、Token 吞吐（输出 / 输入 / 缓存 明细）—— 每 3 秒刷新。

🤖

会变表情的机器人

眨眼、东张西望、蹦跳、冒星星；空闲时打盹飘 zzz。

⚡

工作 / 空闲

交叉比对运行中的 claude、codex 进程与其会话文件，判断谁在忙。

🔒

100% 本地

只读 ~/.claude 和 ~/.codex，数据永不离开你的电脑。

🎛️

两种形态 + 无刘海

刘海下方的药丸、横跨刘海的长条，或无刘海 Mac 上的一条平铺长条。

🖱️

随处可控

组件里的 ⋯ 按钮和菜单栏图标：切换形态、显示隐藏、检查更新、退出。
