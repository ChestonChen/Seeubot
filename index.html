它能做什么

看得见的 AI 会话。

所有数据都从 Claude Code、Codex 和 Cursor 本地写好的文件读取。不联网、无需 API Key、无任何遥测。

实时看板

会话数、工作 vs 空闲、Token 吞吐（输出 / 输入 / 缓存 明细）—— 每 1 秒刷新。

会变表情的机器人

眨眼、东张西望、蹦跳、冒星星；空闲时打盹飘 zzz。

工作 / 空闲

交叉比对运行中的 claude、codex 进程，以及 Cursor 本地会话记录，判断谁在忙。

100% 本地

只读 ~/.claude、~/.codex 和 ~/.cursor 等本机数据，数据永不离开你的电脑。

两种形态 + 无刘海

刘海下方的药丸、横跨刘海的长条，或无刘海 Mac 上的一条平铺长条。

随处可控

组件里的 ⋯ 按钮和菜单栏图标：切换形态、显示隐藏、检查更新、退出。

原理

只读你磁盘上已有的数据。

Claude Code
各 assistant 消息的 usage 累加，按 message.id 去重。
Codex
每个 rollout 最后一条累计 token_count。
Cursor
读取本机 agent transcripts 识别会话，并从本机 usage cache 读取今日 token 统计。
实时会话
Claude / Codex 使用 ps + lsof，Cursor 使用最近活跃的本地 transcript。
工作中
Claude / Codex 按会话文件写入判断；Cursor 按打开中的 turn 和最近 transcript 写入判断。
